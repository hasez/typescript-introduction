## 環境構築

### Node.js のインストール

- install node
```zsh
brew install node@18
```

- update path

```
export PATH="/usr/local/opt/node@18/bin:$PATH"
```

- check install

```
node -v
v18.12.1
```

### TypeScript コンパイラのインストール

- install typescript

```
npm install -g typescript
```

- check install

```
tsc -v
Version 4.9.3
```
